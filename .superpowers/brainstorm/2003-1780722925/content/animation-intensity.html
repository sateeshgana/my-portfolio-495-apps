How intense should the animations be?

All options use the Cyber/Neon Glow theme. The difference is how much motion fills the page.

A

Cinematic — Full spectacle

Particle/star field in hero, animated glowing grid lines, text typewriter reveal on load, neon border pulse on every card, floating orb blobs, smooth parallax scroll. Feels like an interactive AI demo reel.

B

Polished — Purposeful motion

Smooth fade+slide-in on scroll (Intersection Observer), subtle neon glow pulse on hero name, hover lift + glow on cards, staggered entry for skill tags. Fast and elegant — no distracting background motion.

C

Subtle — Motion as polish

Only CSS transitions: hover states, border glows, colour shifts. Section reveals with a simple fade. Performant on all devices, zero distraction, animations are felt not seen.
